# BLOG USING CAPYBARA BDD

#### STEPS TO INSTALL

1. Clone the repo:  

        $ git clone git@github.com:denialtorres/CbBlog.git

2. Install the gems:

        $ bundle install
        
3. Create the db: 
  
        $ rails db:create
    
4. Run the migrations: 

        $ rails db:migrate
     
5. Run the server: 

        $ rails s

#### RUN THE TEST

        $ rspec spec/features/
